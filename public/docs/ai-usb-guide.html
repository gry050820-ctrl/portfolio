1

插 U 盘，双击启动

需要你点一下 不到 10 秒
  • ① 把 U 盘插到电脑上
  • ② 打开 U 盘，找到 ai-usb-installer 文件夹
  • ③ 双击红箭头指着的 「AI一键部署」 文件
不管什么电脑、U 盘是什么盘符，都能正常运行——安装器已经帮你处理好了。
2

打开安装器，看到欢迎页

需要你点一下 约 30 秒
  • ① 双击后会看到这个欢迎界面（如左图）
  • ② 顶部显示 6 个步骤，告诉你接下来要做什么
  • ③ 点右下角 「下一步」，开始环境检测
如果电脑已经有这些工具，安装器会自动跳过，不会重复安装或覆盖你的东西。
3

安装 Claude Code 全家桶

点安装就行 约 2 分钟
  • ① 按照安装器引导，一步步点 「安装」
  • ② 会依次装好：CC Switch → Claude Code → VS Code 扩展
  • ③ 看到进度条跑完、打勾，这一步就完成了
中文界面、注册绕过、模型配置——这些烦人的事安装器都替你做了，不需要手动改任何文件。
4

配置 API Key

需要你操作 约 2 分钟
  • ① 去 DeepSeek 官网（platform.deepseek.com）注册账号
  • ② 进入「API Keys」页面，点「创建 API Key」，点复制（如上图）
  • ③ 回到安装器，在 API Key 配置页粘贴进去，选 DeepSeek，点确定
Key 是你的私人凭证，需要你自己申请。新用户注册有免费额度，正常写代码一个月花不了几块钱。Key 只存在你的电脑上，不会上传。
5

打开 VS Code，开始编程

🎉 完成了
  • ① 安装器会自动打开 VS Code，Claude Code 扩展已经就绪
  • ② 打开终端，输入 claude 就能跟 AI 对话写代码了
如果界面还是英文，或者想切换 AI 模型——看下面的两个小技巧。
📌 手动安装中文语言包（避免自动安装出 bug）
📌 在对话框输入 @switch-model <deepseek-v4-pro> 换模型
